Example Explanation

const numbers = [1, 2, 3, 4, 5]; const doubled = numbers.map(n => n * 2); const evens = numbers.filter(n => n % 2 === 0); const sum = numbers.reduce((acc, n) => acc + n, 0);

🤖 AI Explanation:

"This code demonstrates common JavaScript array operations. First, it creates an array with numbers 1 through 5. Then it uses map() to create a new array where each number is doubled. Next, filter() creates an array containing only even numbers. Finally, reduce() calculates the sum of all numbers in the array. These methods are fundamental for data manipulation in JavaScript."
