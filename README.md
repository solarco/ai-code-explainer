# Privacy Policy – AI Code Reader

Last updated: March 2026

AI Code Reader is a Chrome extension designed to help users understand code by generating explanations using AI models.

## Information We Collect

This extension does **not collect or store personal information**.

The extension may process the following data temporarily:

* Code snippets selected by the user
* API keys provided by the user for AI services
* User-selected AI model preferences

## How Information Is Used

When the user requests a code explanation, the selected code snippet is sent to the chosen AI provider to generate a response.

Supported providers may include:

* OpenRouter
* Google Gemini

The code snippet is transmitted only for the purpose of generating an explanation.

## Data Storage

The extension stores the following locally in the user's browser using Chrome storage:

* API keys for AI providers
* User model selection preferences

This data is stored **locally on the user's device** and is **not transmitted to any server other than the selected AI provider**.

## Data Sharing

This extension does **not sell, transfer, or share user data** with third parties.

The only external communication occurs when sending the user-selected code snippet to the selected AI provider to generate an explanation.

## Security

API keys and preferences are stored using Chrome's secure extension storage system.

Users are responsible for keeping their API keys secure.

## Changes to This Policy

This privacy policy may be updated periodically. Updates will be reflected on this page.
